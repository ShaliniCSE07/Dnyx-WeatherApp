# Dnyx Weather App

A weather application built using HTML, TailwindCSS and Javascript

## Features

- Search for any city  
- View current weather and 5-day forecast  
- Toggle between Celsius (°C) and Fahrenheit (°F)    

## Tech Stack

- HTML  
- JavaScript  
- TailwindCSS
